## 2026-02-24 - Header Flow Name Editing
**Learning:** Hidden interactive elements (like the flow name heading) should have visual indicators (like a pencil icon on hover) to improve discoverability. For accessibility, elements that trigger an action but aren't native buttons must have `role="button"`, keyboard support (Enter/Space), and an `aria-label`.
**Action:** Always add a visual cue and proper ARIA roles/keyboard support when making non-button elements interactive.
