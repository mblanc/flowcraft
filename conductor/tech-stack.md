# Technology Stack - FlowCraft

## Frontend

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **UI Library:** React 19, Tailwind CSS 4
- **Diagramming:** @xyflow/react
- **State Management:** React Context & Zustand
- **Components:** Radix UI primitives

## Backend

- **Platform:** Node.js (Next.js Serverless Functions)
- **Authentication:** NextAuth.js (v5 beta)
- **AI SDK:** @google/genai (Vertex AI integration)
- **Database:** Google Cloud Firestore
- **Storage:** Google Cloud Storage (GCS)

## Infrastructure

- **Containerization:** Docker
- **Hosting:** Google Cloud Run
- **CI/CD:** Google Cloud Build

## AI Models

- **Text:** Gemini 2.0 Flash, Gemini 2.5 Flash
- **Image:** Imagen 4.0, Gemini 2.5 Flash (Image generation)
- **Video:** Veo 3.1 (Fast & Standard)
