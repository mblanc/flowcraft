The **Canvas Workspace** is a freeform creative board where you collaborate with an **AI Director** using natural language chat. Unlike the Flow Editor, which requires you to wire nodes manually, the Canvas works like an interactive whiteboard: you talk to the AI, it builds a production plan, and when you approve, it generates and places media directly on the board.

## Canvas Board

The board is a zoomable, infinite 2D whiteboard. Everything the Director generates appears here as a **card**. There are four card types:

| Card Type      | What it holds                                                      |
| -------------- | ------------------------------------------------------------------ |
| **Image card** | Generated still images (t2i, i2i, upscale)                         |
| **Video card** | Generated video clips (i2v, t2v, concat) with inline playback      |
| **Audio card** | Generated music or speech tracks (t2m, t2s) with a waveform player |
| **Text card**  | Markdown documents — scenario briefs, synopsis, director notes     |

**Board interactions:**

- **Pan**: Drag the background
- **Zoom**: Scroll wheel or trackpad pinch
- **Select**: Click a card to select it
- **Move**: Drag a selected card to reposition it
- **Fullscreen**: Double-click a card to view it at full size
- **Multi-select**: Drag a selection box across multiple cards

### Card Context Menu

Right-click any card to open its context menu:

| Action          | Description                                                            |
| --------------- | ---------------------------------------------------------------------- |
| **Rename**      | Edit the card's display label                                          |
| **Copy Label**  | Copy the label text to clipboard                                       |
| **Copy Prompt** | Copy the generation prompt to clipboard (image/video/audio cards only) |
| **Delete**      | Remove the card from the canvas                                        |

## Left Toolbar

A small vertical toolbar sits on the left edge of the board:

| Button                         | Action                                                                                                                        |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| **Text node** (Type icon)      | Place a blank text card on the board for notes or briefs                                                                      |
| **Import media** (Upload icon) | Upload a local image or video file as a card; uploaded files can be referenced by the Director as style or subject references |

---

## Chat Panel

The chat panel on the left is where all interaction happens. It displays the full conversation history including:

- **Your messages** with any file attachments you've added
- **Director responses** with streamed text and a collapsible thought log showing the Director's reasoning before it responds
- **Plan approval cards** when the Director has built a production plan (see [Managing Production Plans](/canvas/topology))
- **Clarification prompts** when the Director needs more information before planning
- **Suggested action buttons** after a generation completes, offering smart next steps

### Chat Input

The input field at the bottom of the chat panel supports:

- **Free text** — describe what you want in plain English
- **`@` mentions** — type `@` to reference a card already on the board; the Director will use that card as a subject or style reference in its plan
- **File attachments** — attach images or videos directly from your computer

## Creative Loop

Every Canvas session follows the same conversational loop:

1. **Prompt** — Type a request in the chat.
2. **Think** — The Director reasons about your request before responding (you see the thought log stream in real time).
3. **Clarify** _(if needed)_ — The Director may ask a multiple-choice clarifying question before building a plan (e.g. "What aspect ratio?" or "Portrait or landscape?"). Pick an option or type a custom answer.
4. **Plan** — The Director presents a **Production Plan** as a card in the chat, listing all the steps it will run and in what order.
5. **Approve** — Review the plan. You can ask the Director to change it, or click **Approve & Execute** to start generation.
6. **Generate** — Steps execute in dependency order, with parallel steps running simultaneously. Cards appear on the board as each step completes.
7. **Refine** — Once generation is done, the Director suggests 2–3 follow-up actions as buttons (e.g. "Animate this", "Try a wider shot", "Add music").
