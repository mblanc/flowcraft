The **Canvas Workspace** (accessed via the **Agents** section in the sidebar) is a freeform creative board where you collaborate with an **AI Director** using natural language chat. Unlike the Flow Editor, which requires you to connect boxes manually, the Canvas behaves like an interactive whiteboard. You talk to the AI, and the AI places, edits, and organizes your images and videos on the board for you.

---

## 🖥️ The Canvas Interface

The Canvas Workspace is divided into two main panels:

```
┌─────────────────────────────────┬─────────────────────────────────┐
│                                 │                                 │
│  AI Director Chat (Left)        │  Canvas Board (Right)           │
│                                 │                                 │
│  💬 "Generate a sci-fi city"    │       ┌───────────────┐         │
│                                 │       │ [City Image]  │         │
│  🤖 "Here is the plan. Approve?"│       └───────────────┘         │
│                                 │                                 │
└─────────────────────────────────┴─────────────────────────────────┘
```

1. **AI Director Chat (Left):** The conversation panel where you guide the AI. You can write prompts, request modifications, ask questions, or click suggested action buttons.
2. **Canvas Board (Right):** A zoomable, infinite 2D whiteboard. Any image or video generated by the AI (or uploaded by you) appears here as a **Media Card**. You can select card layers, drag them to rearrange your board, or double-click to view them in fullscreen.

---

## 🔄 How the Creative Loop Works

Creating in the Canvas Workspace follows a simple, conversational loop:

1. **You Prompt:** Type a request in the chat (e.g. _"I want a movie scene showing a spaceship landing in a desert"_).
2. **AI Plans:** The AI Director processes your prompt, refines it using built-in prompt engineering, and returns a **Production Plan** in the chat.
3. **You Approve:** Review the plan. You can ask the AI to change aspects of it (e.g. _"Make the desert look like Mars"_), or simply click **Approve & Execute**.
4. **AI Generates:** The AI executes the steps, generating the images or videos and placing them directly onto the Canvas Board.
5. **You Refine:** Click on any generated card to upscale it, animate it into a video, or request variations.
