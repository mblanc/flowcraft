Welcome to **FlowCraft**, your ultimate visual playground for AI-powered media creation! Whether you are a writer, designer, video creator, or marketer, FlowCraft empowers you to create stunning images, cinematic videos, and structured text using Google's state-of-the-art Gemini AI models (including Gemini, Imagen, and Veo).

## 🚀 Two Ways to Create

### 1. Flow Editor (Automated Pipelines)

If you like structure, automation, and repeatibility, the **Flow Editor** is for you. It provides a visual drag-and-drop workspace where you connect specialized blocks (nodes) together.

- **How it works:** You wire up input files, LLM prompts, image generators, and video upscalers into a pipeline.
- **Why use it:** Run batch processes (e.g., generate 10 variations of an image with different prompt styles), automate content generation, or create complex multi-step workflows.

### 2. Canvas Workspace (Interactive AI Chat)

If you prefer freeform exploration and collaborating with an AI partner, choose the **Canvas Workspace**. It is a visual board where you chat with an **AI Director** who helps you brainstorm, plan, and generate media assets.

- **How it works:** You chat with the AI Director in plain English. The Director creates visual production cards, arranges them on the board, and executes them when you approve.
- **Why use it:** Storyboarding, creative brainstorming, interactive design, and building scenes organically through natural conversation.

---

## ✨ Features at a Glance

- **Multi-Modal Generation:** Create high-quality text, realistic images, cinematic videos, and custom music/audio tracks in one place.
- **Batch Processing:** Run workflows on multiple inputs simultaneously to speed up your creative pipeline.
- **Visual Connections:** Use drag-and-drop links to pass outputs from one block (like a text prompt) to another (like an image generator).
- **AI-Guided Help:** Work alongside an AI agent that handles prompt enrichment, dependency management, and asset coordination.
