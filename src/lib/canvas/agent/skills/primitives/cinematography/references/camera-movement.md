## Camera Movement (video only)

Never combine two moves in a single video node — it causes jitter in every model. Pick one move + one rhythm word.

Rhythm words: `gradual`, `gentle`, `smooth`, `abrupt`, `sharp`, `fluid`

| Movement                    | Description                                        | Typical use                               |
| --------------------------- | -------------------------------------------------- | ----------------------------------------- |
| Static / tripod             | No movement — camera locked                        | Stability, formality, quiet observation   |
| Pan                         | Camera rotates horizontally on fixed axis          | Following action, revealing environment   |
| Tilt                        | Camera rotates vertically on fixed axis            | Revealing height, power dynamics          |
| Dolly in / tracking in      | Camera physically moves toward subject             | Intimacy, revelation, building tension    |
| Dolly out / pull-back       | Camera physically moves away from subject          | Isolation, context reveal, release        |
| Dolly zoom (Vertigo effect) | Lens zooms while dolly moves in opposite direction | Psychological disorientation, vertigo     |
| Arc shot                    | Camera circles around the subject                  | Heroic reveal, 360° environment survey    |
| Crane shot                  | Camera rises or descends on a crane                | Grand reveal, overview, lofty transitions |
| Drone shot                  | Aerial and mobile — distinct from fixed aerial     | Landscape, chase sequences, scale         |
| Handheld                    | Intentionally unstable, operator-held              | Immediacy, tension, documentary feel      |
| Steadicam                   | Smooth moving shot, gimbal-stabilized              | Fluid immersive following of action       |
| Whip pan                    | Extremely fast horizontal pan                      | Transition between shots, disorientation  |

### Prompt phrasing

Format: `[rhythm word] [movement name].`

- "Gradual dolly push-in."
- "Static tripod shot, no movement."
- "Smooth steadicam following the subject."
- "Fluid crane shot rising above the rooftop."
- "Gentle pan left, revealing the cityscape."
