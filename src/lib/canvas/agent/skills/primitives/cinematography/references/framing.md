## Framing / Subject Size

| Shot Type         | Abbrev    | What it shows                                       | Use when                                           |
| ----------------- | --------- | --------------------------------------------------- | -------------------------------------------------- |
| Extreme close-up  | ECU / XCU | Single detail — eyes, lips, texture, object surface | Maximum intimacy or object detail; hiding identity |
| Close-up          | CU        | Face/head, or a specific object                     | Emotion, identity, recognition                     |
| Medium close-up   | MCU       | Chest/shoulders up                                  | Conversations, interviews, "news anchor"           |
| Medium shot       | MS        | Waist up                                            | Dialogue, gestures, character interaction          |
| Cowboy shot       | —         | Mid-thigh up                                        | Action near hands or holsters; Western register    |
| Medium long shot  | 3/4 shot  | Knees up                                            | Body language + environment balance                |
| Full shot         | FS / LS   | Entire body head-to-toe                             | Costume, posture, full-body action                 |
| Wide shot         | WS        | Subject + significant environment                   | Relationship to place, context around action       |
| Extreme wide shot | EWS       | Subject tiny, landscape-dominant                    | Isolation, epic scale, geography                   |
| Establishing shot | —         | Location/context, often wide                        | Opens a scene; communicates place and time         |
