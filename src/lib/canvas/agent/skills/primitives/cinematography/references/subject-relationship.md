## Subject Relationship / Framing

| Shot Type               | Notes                                                                 |
| ----------------------- | --------------------------------------------------------------------- |
| Single                  | One subject, full attention on them                                   |
| Two-shot                | Two subjects — conversations, relationships, conflict                 |
| Three-shot              | Three subjects — group dynamics                                       |
| Group shot              | Four or more subjects                                                 |
| Over-the-shoulder (OTS) | Subject A seen past Subject B's shoulder — dialogue standard          |
| Reverse shot            | Opposite angle of a previous OTS — pairs with OTS in dialogue editing |
| POV                     | Camera represents a character's literal eyeline                       |
| Reaction shot           | Subject responding to off-screen or on-screen action                  |
| Insert shot             | Tight cut-in on a detail (clock, hand, document, object)              |
| Cutaway                 | Cut to something related but outside the main scene action            |
| Master shot             | Wide uninterrupted shot of an entire scene — the "safety" coverage    |
| Talking head            | Direct-to-camera, usually MCU — docs, interviews, mockumentaries      |
| B-roll                  | Supplemental footage cut over dialogue or narration                   |
