# Top 10 Demo Flows for GenMedia Node-Based Tool

## Selection Criteria

✓ **Immediate "wow" factor** - Visually impressive results  
✓ **Clear value proposition** - Solves real production problems

## 💡 Implementation Tips

### For Each Demo Flow:

- **Include sample inputs** - Provide ready-to-use starting prompts/images
- **Show before/after** - Make the transformation clear
- **Add tooltips** - Explain what each node does
- **Enable one-click run** - Let users execute the entire flow instantly
- **Allow customization** - Make prompts/parameters easy to modify
- **Export templates** - Let users save and reuse successful flows

### Node Connections to Highlight:

- Text → Image (generation from scratch)
- Image → Image (transformation/editing)
- Image → Video (bringing stills to life)
- Any → Upscale (quality enhancement)
- Branching (one input → multiple outputs for A/B testing)
- Sequential (storyboard/multi-shot sequences)

### Success Metrics:

- **Clarity**: Can a newcomer understand the flow in 10 seconds?
- **Inspiration**: Does it make them think "I want to try this"?
- **Practicality**: Can they immediately see business/creative value?
